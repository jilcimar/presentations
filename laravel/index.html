Como configurar o Laravel?

Precisamos apenas de 1 script no terminal

SETUP (LARAVEL)

  • sudo apt-add-repository ppa:ondrej/php
  • sudo apt-get update
  • sudo apt-get install php-mbstring
  • sudo apt-get install composer -y
  • sudo apt-get install php7.0-pgsql -y
  • sudo apt-get install apache2 libapache2-mod-php7.0 -y
  • sudo apt-get install php7.0-dev -y
  • sudo apt-get install php7.0-zip -y
  • sudo apt-get install php7.0-xml -y
  • sudo apt-get install php7.0-mbstring -y
  • sudo apt-get install php7.0-cgi -y
  • sudo apt-get install php7.0-curl -y
  • sudo apt-get install php7.0-zip

SETUP (BANCO DE DADOS)

  • sudo apt-get install postgresql pgadmin3
  • sudo su postgres
  • psql
  • ALTER USER postgres WITH PASSWORD ‘pgadmin’;
